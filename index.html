Curso Técnico em Informática

EMTI - Ensino Médio em Tempo Integral

📘 Sobre o Curso

🎯 Objetivo

Formar profissionais capazes de atuar nas áreas de programação, manutenção, redes, web e robótica, com postura ética e visão crítica.

🗂️ Estrutura Curricular

📘 1º Ano

  • Informática Básica
  • Lógica de Programação

📗 2º Ano

  • Arquitetura de Computadores
  • HTML/CSS
  • Sistemas Operacionais
  • Gestão do Tempo
  • Programação Orientada a Objetos (Java)
  • Programação Web I (JS, PHP, MySQL)
  • Robótica

📙 3º Ano

  • Redes de Computadores
  • Design para Web
  • Banco de Dados e CMS
  • Programação Web II
  • Laboratório de Hardware
  • Laboratório de Software
  • Projeto Final Web

📘 About the Course

🎯 Objective

Train professionals capable of working in programming, maintenance, networks, web development and robotics, with ethics and critical thinking.

🗂️ Curriculum Structure

📘 Year 1

  • Computer Basics
  • Programming Logic

📗 Year 2

  • Computer Architecture
  • HTML/CSS
  • Operating Systems
  • Time Management
  • Object-Oriented Programming (Java)
  • Web Programming I (JS, PHP, MySQL)
  • Robotics

📙 Year 3

  • Computer Networks
  • Web Design
  • Database and CMS
  • Web Programming II
  • Hardware Lab
  • Software Lab
  • Capstone Web Project
