📘 1º Ano

🖥️ Informática Básica

Objetivo: Aplicar fundamentos da informática.

  • Evolução da Informática
  • Binário, Unidades de medida
  • Hardware e Software
  • Redes, Virtualização, Nuvem

⚙️ Lógica de Programação

Objetivo: Criar algoritmos e programas estruturados.

  • Algoritmos, Variáveis
  • Estruturas de decisão e repetição
  • Teste de mesa e boas práticas

📗 2º Ano

🧰 Arquitetura e Manutenção

Objetivo: Planejar e executar montagem de computadores.

  • Arquitetura, Processadores
  • Memórias, Armazenamento
  • Placa-mãe, Periféricos

🌐 HTML / CSS

Objetivo: Criar páginas web com HTML5 e CSS3.

  • HTML semântico, formulários
  • CSS3: seletores, layout
  • Responsividade e acessibilidade

🕰️ Gestão do Tempo

Objetivo: Organizar e priorizar tarefas com eficiência.

  • Planejamento e matriz de tempo
  • Ferramentas de produtividade

💽 Sistemas Operacionais

Objetivo: Instalar e configurar sistemas em desktops e mobile.

  • Instalação e gerenciamento
  • Virtualização e manutenção

☕ POO (Java)

Objetivo: Programar com conceitos orientados a objetos.

  • Classes, métodos, atributos
  • Herança, polimorfismo

🕸️ Programação Web I

Objetivo: Criar aplicações web com backend e banco de dados.

  • PHP, JavaScript, MySQL
  • Formulários e persistência

🤖 Noções de Robótica

Objetivo: Entender sensores, atuadores e microcontroladores.

  • Arduino, Simulide
  • Programação básica de robôs

📙 3º Ano

📡 Redes de Computadores

Objetivo: Instalar e configurar redes locais.

  • Topologias, protocolos
  • Crimpagem, cabeamento

🎨 Design

Objetivo: Criar interfaces visuais para web.

  • Composição, tipografia, cores
  • Wireframes, protótipos

🗂️ Banco de Dados e CMS

Objetivo: Gerenciar conteúdo web e dados.

  • WordPress, phpMyAdmin
  • SQL e segurança

💻 Programação Web II

Objetivo: Desenvolver sistemas com segurança e persistência.

  • APIs REST, cookies, sessões
  • CRUD e SQL seguro

🔧 Laboratório de Hardware

Objetivo: Executar manutenção preventiva e corretiva.

  • Instalação de peças, diagnósticos

🧪 Laboratório de Software

Objetivo: Corrigir, restaurar e configurar sistemas operacionais.

  • Backup, scripts, linha de comando

📁 Projeto Final

Objetivo: Desenvolver um sistema web completo em equipe.

  • Planejamento, execução e entrega
