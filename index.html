💾Curso Técnico em Informática

📚 1º Ano

🧩 Pensamento Computacional

  • Raciocínio lógico e resolução de problemas
  • Decomposição, abstração e reconhecimento de padrões
  • Algoritmos do cotidiano e uso da lógica na tomada de decisão
  • Relacionamento entre o pensamento computacional e outras áreas do conhecimento

💡 Informática Básica

  • Evolução da informática e história dos computadores
  • Representação binária, unidades de medida e codificação de dados
  • Hardware, software, sistemas operacionais, aplicativos
  • Conceitos de internet, redes e computação em nuvem

🧠 Lógica de Programação

  • Algoritmos, fluxogramas e pseudocódigos
  • Variáveis, constantes, operadores, expressões
  • Condicionais, estruturas de repetição, teste de mesa
  • Boas práticas de programação

📚 2º Ano

🔧 Arquitetura e Manutenção de Computadores

  • Componentes físicos: placa-mãe, memória, processador, armazenamento
  • Montagem, manutenção preventiva e corretiva
  • Procedimentos de segurança e normas técnicas

🌐 HTML / CSS

  • Estrutura HTML, semântica, formulários, acessibilidade
  • CSS interno e externo, seletores, layouts, responsividade

💻 Sistemas Operacionais

  • Instalação e configuração de sistemas Windows e Linux
  • Interface gráfica, linha de comando, permissões
  • Virtualização e manutenção básica

☕ Programação Orientada a Objetos (Java)

  • Conceitos de classe, objeto, herança, polimorfismo
  • Estrutura básica de programas em Java

🌍 Programação Web (JavaScript, PHP, MySQL)

  • Front-end com JavaScript, formulários e interatividade
  • Introdução ao PHP, conexões com banco de dados

⏳ Gestão do Tempo

  • Planejamento pessoal e profissional
  • Ferramentas e técnicas para produtividade

🤖 Noções de Robótica

  • Sensores, atuadores, microcontroladores (Arduino)
  • Montagem e programação básica

📚 3º Ano

🌐 Redes de Computadores

  • Topologias, protocolos, cabeamento, conectividade
  • Instalação de redes locais

🎨 Design

  • Imagem digital, composição, cores, tipografia
  • Wireframes e prototipação

🗂️ Gerenciador de Conteúdo / Banco de Dados

  • Uso de CMS (WordPress), hospedagem e publicação
  • Banco de dados relacional, SQL básico

🧪 Programação Web II

  • Integração front e back-end via APIs
  • CRUD, sessões, cookies, segurança (SQL Injection)

🔌 Laboratório de Hardware

  • Reparos técnicos, diagnóstico e solução de falhas
  • Normas e ferramentas profissionais

💾 Laboratório de Software

  • Imagens de sistema, backups, restauração, automação
  • Comandos avançados, permissões e scripts

🧱 Laboratório Web (Projeto Integrador)

  • Planejamento e desenvolvimento de sistemas
  • Integração entre linguagens e banco de dados

💾Technical Course in Informatics

📚 1st Year

🧩 Computational Thinking

  • Logical reasoning and problem solving
  • Decomposition, abstraction, and pattern recognition
  • Everyday algorithms and logic in decision making
  • Connections between computational thinking and other disciplines

💡 Basic Informatics

  • History and evolution of computing
  • Binary representation, data measurement units and encoding
  • Hardware, software, operating systems, applications
  • Internet, networks and cloud computing concepts

🧠 Programming Logic

  • Algorithms, flowcharts and pseudocode
  • Variables, constants, operators, expressions
  • Conditionals, loops, desk checking
  • Best programming practices

📚 2nd Year

🔧 Computer Architecture and Maintenance

  • Hardware components: motherboard, memory, processor, storage
  • Assembly, preventive and corrective maintenance
  • Safety procedures and technical standards

🌐 HTML / CSS

  • HTML structure, semantics, forms, accessibility
  • Internal and external CSS, selectors, layouts, responsiveness

💻 Operating Systems

  • Installation and configuration of Windows and Linux systems
  • GUI, command line, permissions
  • Virtualization and basic maintenance

☕ Object-Oriented Programming (Java)

  • Concepts of class, object, inheritance, polymorphism
  • Basic structure of Java programs

🌍 Web Programming (JavaScript, PHP, MySQL)

  • Front-end with JavaScript, forms and interactivity
  • Introduction to PHP, database connections

⏳ Time Management

  • Personal and professional planning
  • Tools and techniques for productivity

🤖 Introduction to Robotics

  • Sensors, actuators, microcontrollers (Arduino)
  • Basic assembly and programming

📚 3rd Year

🌐 Computer Networks

  • Topologies, protocols, cabling, connectivity
  • Installation of local networks

🎨 Design

  • Digital image, composition, colors, typography
  • Wireframes and prototyping

🗂️ CMS / Database

  • Using CMS (WordPress), hosting and publishing
  • Relational database, basic SQL

🧪 Web Programming II

  • Front-end and back-end integration via APIs
  • CRUD, sessions, cookies, security (SQL Injection)

🔌 Hardware Lab

  • Technical repairs, diagnostics and troubleshooting
  • Professional tools and safety standards

💾 Software Lab

  • System images, backup, restore, automation
  • Advanced commands, permissions and scripts

🧱 Web Lab (Capstone Project)

  • System planning and development
  • Integration of languages and database
